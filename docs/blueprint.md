# **App Name**: BalanceView

## Core Features:

- Income Input: Allow the user to input their monthly income.
- Bill Input: Allow the user to input recurring bills (mortgage, utilities, etc.) with amounts.
- Balance Calculation: Calculate the remaining balance after deducting bills from income.
- Pie Chart Visualization: Display a pie chart visualizing the proportion of income allocated to each bill and the remaining balance.
- Currency Formatting: Format all monetary values in Â£ (GBP).

## Style Guidelines:

- Primary color: Light blue (#ADD8E6), creating a soft, calm, and approachable interface. This color evokes feelings of trust and clarity.
- Background color: Very light gray (#F0F0F0), a desaturated variant of the primary hue. Provides a clean, modern, neumorphic feel.
- Accent color: Muted green (#8FBC8F), for a gentle contrast that highlights key information, such as available balance. This aligns with the app's focus on financial well-being.
- Body and headline font: 'PT Sans', a humanist sans-serif font known for its modern look combined with a little warmth.
- Use simple, outlined icons to represent different bill categories and actions, following the neumorphic aesthetic.
- Employ a clean and minimal layout with soft shadows and rounded corners to achieve a neumorphic design. Prioritize readability and ease of use.
- Incorporate subtle transitions and animations for user interactions, providing feedback and enhancing the overall user experience without being distracting.
